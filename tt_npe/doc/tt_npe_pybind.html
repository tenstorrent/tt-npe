class Transfer (...)

Represents a contiguous series of packet(s) from a single source to one or more destinations. Roughly 1:1 equivalent to any single call to dataflow_api.h:noc_async_(read|write).

init(self: tt_npe_pybind.Transfer, packet_size: int, num_packets: int, src: tt_npe_pybind.Coord, dst: tt_npe_pybind.Coord, injection_rate: float, phase_cycle_offset: int, noc_type): tt_npe_pybind.NocType) -> None

Creates a new, fully-initialized npe.Transfer object. Arguments required are (in order):

  1. packet_size : Size of packet(s) being transferred in bytes. Must be greater than 1.

  2. num_packets : Number of packets to be transferred. Must be 1 or greater.

  3. src : npe.Coord specifying {row,col} location of the source of the transfer.

  4. dst : npe.Coord specifying {row,col} location of the destination of the transfer.

  5. injection_rate : (inferred by default, can leave as 0). Override rate at which packets are injected into network from source in GB/s.

  6. phase_cycle_offset : Earliest possible cycle transfer can start, relative to the start of its containing npe.Phase.

  7. noc_type : The noc type used (NOC_0,NOC_1).
